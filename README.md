## 📌 MD# — Markdown-based Programming Language (F#)

Экспериментальный интерпретируемый язык программирования с синтаксисом на основе Markdown.
Разработан как учебный проект по функциональному программированию и компиляторам.

---

## 🚀 Основная идея

Язык использует Markdown-структуру (списки, отступы) как синтаксис для описания программ.
Это позволяет исследовать альтернативные подходы к представлению кода и его парсингу.

---

## 🧠 Возможности языка

* Объявление переменных
* Выражения и вычисления
* Пользовательские функции
* Базовая модель выполнения (interpreter-based execution)

---

## ⚙️ Архитектура

Процесс выполнения программы:

```
Markdown source → Parser → AST → Interpreter → Result
```

### Компоненты:

* **Parser** — разбор Markdown в AST
* **AST (Abstract Syntax Tree)** — представление программы
* **Interpreter** — выполнение программы

---

## 🧰 Стек

* F#
* Functional programming
* Recursive parsing

---

## 🔍 Пример программы

```md
- square = >[!] x
  > x * x
  > ^x

- result = square 5
```
